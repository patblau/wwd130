WWD130 – Project Index

All assignments in one place. Click a card to open that project.

How to add a new assignment:
  1. Create a new folder at the repo root (e.g., week05_typography/).
  2. Put that project’s index.html inside the folder.
  3. Edit projects.json and add a new entry pointing to your folder (see below).
  4. Commit & push. The card will appear automatically.
